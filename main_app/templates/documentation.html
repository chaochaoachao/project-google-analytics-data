Versions required for Python files:

  • Flask==1.1.1
  • google-cloud-bigquery==1.24.0
  • SQLAlchemy==1.3.11
  • pybigquery==0.4.13
  • pandas==0.24.2
  • plotly==4.3.0

    • Machine Learning Algorithms

    KMeans from scikit-learn was used to find the customer clusters.

    Facebook Prophet was used to forecast the total revenue for the next 30 days.

    Microsoft LightGBW was used to predict revenue at the user level.
